# 📝 Advanced Todo App

A full-featured Todo application with authentication, task workflow, and search functionality.

## 🚀 Features

- 🔐 User Login/Signup with validation
- ✅ Create, Edit, Delete todos
- 🔄 Three-column workflow (Todo → Inprogress → Done)
- 🔍 Search by title or description
- 🏷️ Priority badges (High/Medium/Low)
- 💾 LocalStorage persistence
- 📱 Responsive design

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- React Hook Form + Yup
- Lucide Icons
- React Hot Toast

npm install
npm start
